Widgets to include: - Live clock with smooth seconds animation - Greeting with editable name + date - Top 3 tasks for today (checkable, editable) - Scratchpad (auto-saves) - Bookmarks / daily links (defaults: LinkedIn, Notion, Gmail, IESE Portal (https://my.iese.edu/), Claude (https://claude.ai), ChatGPT (https://chat.openai.com), Gemini (https://gemini.google.com)) - Countdown card (default event: "End of Year 1", default date: May 30, 2026; fully editable) - Mini habit tracker (3 habits with daily checkboxes) - Daily quote (rotates from a list) Animations: - Cards fade and slide up on load, staggered - Card lift + glow on hover using accent colors from the design system - Quote fades in smoothly - Checked tasks get a strikethrough animation - Background has a very slow ambient drift using the design system's primary colors Apply the design system consistently across all cards, typography, spacing, and colors. Add a live news feed to your bento

Claude Code

Open Claude Code inside your bento folder, then paste this prompt

Prompt
Read the existing index.html, script.js, and style.css. Then do the following: 1. Add a news card to the bento. Add a new card to index.html: a "Live News" section with 10 headline items (title + source). In script.js, add a fetchNews() function that calls http://localhost:3000/api/news and populates the card on load, refreshing every 1 minute. 2. Build the backend. Create server/server.js — an Express server on port 3000 that fetches the latest headlines from these 7 RSS feeds using rss-parser: BBC News, TechCrunch, Hacker News, Reuters, The Guardian, Wired, and Ars Technica. On startup, print each headline to the terminal with chalk: source in color, headline text, time posted. Expose /api/news returning JSON. Enable CORS. 3. Create server/package.json with dependencies: express, rss-parser, chalk, cors. 4. Create start.bat at the root that: - Starts the server (cd server && npm install && node server.js) in the background - Waits 2 seconds for the server to boot - Automatically opens index.html in the default browser
💡 Claude Code reads your existing files and integrates cleanly — it won't break what Antigravity built. Once done, double-click start.bat and watch the terminal light up.

3 rules to take home

01
Be specific The more detail in your prompt, the better the output. Vague prompts produce generic results.
02
Iterate, don't regenerate First output is never final. Keep prompting — each round gets closer to what you actually want.
03
You're the editor AI builds fast, but you decide what stays. Think of it as a very fast intern, not a replacement.
